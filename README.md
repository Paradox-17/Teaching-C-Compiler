# 教学型 C 语言编译器

用于编译原理实验课的 C 语言子集编译器，包含词法分析、语法分析、语义分析、汇编代码生成及虚拟机。

## 特性

- 支持 `int` / `float` 变量声明与赋值
- 支持 `if-else`、`while` 控制流
- 支持 `scanf` / `printf`（`%d`, `%f`）
- 图形界面（Tkinter）
- 可查看 AST 与汇编代码

## 使用说明

1. 将待编译的 C 语言程序保存为 `.txt` 文件，**放在与 `Compiler.py` 相同的目录下**
2. 运行 `python Compiler.py`
3. 在 GUI 中输入文件名（如 `EX1`），点击“开始编译”

> 本仓库提供了 13 个示例程序（EX1~EX13），可直接使用。

## 示例程序说明

本仓库提供了 13 个示例程序（EX1~EX13），覆盖正确语法与常见错误类型：

| 文件 | 用途 |
|------|------|
| EX1 | 变量声明 |
| EX2 | 变量赋值 |
| EX3 | 表达式运算 |
| EX4 | 分支语句 |
| EX5 | 循环语句 |
| EX6 | 函数声明与调用 |
| EX7 | 标准输入输出 |
| EX8~EX9 | 变量作用域管辖 |
| EX10 | 词法错误（非法字符、错误数字格式） |
| EX11 | 语法错误（缺少分号、括号不匹配等） |
| EX12 | 语义错误（变量未声明、重复定义等） |
| EX13 | 综合案例（素数筛选） |

> 正确程序可直接编译运行；错误程序用于测试编译器的错误诊断能力。

## 运行环境

- Python 3.8 或更高版本
- 无需额外安装库（仅使用标准库 `tkinter`）
